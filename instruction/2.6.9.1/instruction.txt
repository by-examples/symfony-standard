Branch:
    2.6.9/Full.1


1. Rebuild branches with components

2. Orphan branch

Create an orphan branch 2.6.9/Full.1 for Symfony Standard v2.6.9

git checkout --orphan 2.6.9/Full.1 v2.6.9
git commit -m "Symfony Standard v2.6.9"

Creating a branch like symfony-standard-v2.6.4 for every release
is pointless as --orphan will yield exactly the same result.


3. Distribution branch

Create distribution-full-2.6.9 branch (purpose: to create a bulb)
This is a temporary branch.

git checkout -b distribution-full-2.6.9 2.6.9/Full.1

5. Update dependencies:

Remove bin directory:

rm -rf bin
rm composer.lock

vagrant up
vagrant ssh

./initialize.bash

sudo composer self-update
composer update

./reload.bash
bin/behat

6. Commit updated dependencies

git snapshot "[STARTUP] Updated dependencies"

Remember to commit app/config/parameters.yml file.

7. Merge distribution-full-2.6.9 branch into 2.6.9/Full.1 branch with a bulb

git checkout 2.6.9/Full.1
git merge --no-ff distribution-full-2.6.9

Merge commit should have the caption:

SymfonyAppStartup 2.6.9.1


8. Remove distribution branch:

git branch -D distribution-full-2.6.9
